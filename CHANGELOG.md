# Changelog

## 0.2.0

- Cycle through multiple tags per cell (default: skip, debug, setup, misc)
- Filter dropdown to select which tags are skipped on Run All
- Add and remove custom tags from the filter dropdown
- "Skip untagged cells" option to only run explicitly tagged cells
- Tags stored as standard Jupyter cell tags for compatibility
- Tag list and skip selections persisted per workspace

## 0.1.0

- Mark code cells to be skipped during "Run All"
- Toggle skip via inline cell toolbar button
- "Run All (Skip Marked)" command in notebook toolbar
- "Skipped on Run All" status bar indicator per cell
- Skip flag persists in standard Jupyter cell metadata
